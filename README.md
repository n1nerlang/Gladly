<p align="center">
  Gladly subverts the traditional, gritty sandbox aesthetic with a clean, welcoming identity. It is built from the ground up to offer a highly performant, accessible block-building experience that runs smoothly on mobile hardware without sacrificing infinite world generation.
</p>

## ✨ Key Features

*   **Mobile-First Architecture:** Highly optimized rendering pipeline and touch-friendly UI tailored for fluid performance on Android devices.
*   **Infinite Terrain Generation:** Procedurally generated biomes, subterranean cave networks, and fluid dynamics.
*   **True Open Source Freedom:** Fully licensed under GPLv3—copy it, modify it, extend it, and share your changes with the world.
*   **Lightweight Engine:** Engineered for fast load times and minimal battery drain, making it perfect for portable play and active on-device testing.
